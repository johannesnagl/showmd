The problem with .md files on macOS #

Quick Look shows raw text

Press Space on a .md file and macOS dumps the raw source — hashes, asterisks, dashes and all. Unreadable.

VS Code is overkill

Opening a full editor just to read a README or AGENTS.md takes seconds and shifts your focus entirely.

Frontmatter is ignored everywhere

Every other viewer dumps the --- block as noise at the top. No app parses it.

Three steps, then forget about it #

  1. 01

    Download & open show.md

    The Quick Look extension registers automatically. No configuration, no Terminal commands.

  2. 02

    Enable in System Settings

    System Settings → Privacy & Security → Extensions → Quick Look. One toggle.

  3. 03

    Spacebar any .md file in Finder

    Rendered markdown, frontmatter table, source toggle. That's it.
